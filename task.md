# DESAFIO: Desenvolvedor Front-End (Pleno/Sênior)

## 📌 Atividade Técnica — Desenvolvedor Front-End (Pleno/Sênior)

---

## 🎯 Objetivo da Atividade

Avaliar:

- Organização de componentes
- Gerenciamento de estado
- Construção de interfaces dinâmicas
- Experiência prática com sistemas administrativos

---

## 🧩 Cenário

Criar uma interface básica de **agenda diária** para:

- Visualização de agendamentos
- Criação de novos agendamentos

---

## ✅ Requisitos Obrigatórios

1. Lista de horários (ex: `08:00 — 18:00`)
2. Exibir agendamentos existentes
3. Botão para criar novo agendamento

---

## 📦 O que deve ser entregue

- Estrutura de componentes
- Integração com dados mockados
- Formulário simples para criação de agendamentos

---

## ⭐ Diferenciais (Opcional)

- Modal para criação de agendamento
- Validações básicas no formulário
- Organização clara de pastas

---

## 🏆 Critérios de Avaliação

- Organização dos componentes
- Simplicidade da solução
- Clareza do código
